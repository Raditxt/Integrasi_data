SellEase API Documentation

GET /dashboard/stats

Menampilkan statistik utama untuk dashboard admin.

Method

GET

Parameters

Tidak ada parameter.

Contoh Request

GET /dashboard/stats HTTP/1.1
Host: your-api.com

Contoh Response (200 OK)

{
  "total_products": 120,
  "total_customers": 340,
  "total_orders": 567,
  "avg_rating": 4.3,
  "total_suppliers": 22,
  "total_employees": 15
}

Contoh Response (500 Error)

{
  "status": "error",
  "message": "penjelasan error dari sistem"
}

Penjelasan Field

Field Tipe Deskripsi
total_productsintTotal produk yang tersedia
total_customersintTotal pelanggan
total_ordersintTotal pesanan
avg_ratingfloatRata-rata rating pembelanjaan
total_suppliersintTotal supplier
total_employeesintTotal karyawan
